# 🍽️ FoodFinder

An Android application that lets users add and discover restaurants, storing data in real-time using Firebase Realtime Database.

---

## 📱 Features

- Add restaurants with name, location, and rating
- Retrieve and display all saved restaurants in real-time
- Firebase Realtime Database integration for cloud storage
- Clean, minimal UI with input fields and action buttons

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Kotlin |
| Platform | Android |
| Database | Firebase Realtime Database |
| Architecture | Single Activity |
| Min SDK | Android 5.0+ (API 21) |

## 🚀 Getting Started

### Prerequisites

- Android Studio (Android Studio Panda 4)
- A Firebase account
- An Android device or emulator (API 21+)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/FoodFinder.git
   cd FoodFinder
   ```

2. **Connect to Firebase**
   - Go to the [Firebase Console](https://console.firebase.google.com/)
   - Create a new project (or use an existing one)
   - Add an Android app with package name `com.example.foodfinder`
   - Download the `google-services.json` file and place it in the `app/` directory

3. **Enable Realtime Database**
   - In the Firebase Console, navigate to **Realtime Database**
   - Click **Create Database** and start in test mode

4. **Open in Android Studio**
   - Open the project and let Gradle sync
   - Run the app on a device or emulator

---

## 🗄️ Data Model

Each restaurant entry stored in Firebase has the following structure:

```json
{
  "Restaurants": {
    "<auto-generated-id>": {
      "name": "Pizza Palace",
      "location": "New York, NY",
      "rating": 4.5
    }
  }
}
```

---

## 🧩 Key Components

### `MainActivity.kt`
- Initializes Firebase Database reference pointing to `"Restaurants"` node
- `addRestaurant()` — Pushes a new `Restaurant` object to Firebase with a unique key
- `readRestaurants()` — Attaches a `ValueEventListener` to read all restaurant entries in real-time and logs them

### `Restaurant.kt` *(data class)*
```kotlin
data class Restaurant(
    val name: String = "",
    val location: String = "",
    val rating: Double = 0.0
)
```
